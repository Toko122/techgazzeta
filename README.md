# Building Management System

Building Management System is a modern web application designed to manage residential buildings and their day-to-day operations.  
The system is built with a service-oriented mindset and focuses on clean UI, usability, and real-world project structure.

The application is **fully responsive** and works smoothly across desktop, tablet, and mobile devices.

---

## Services

The system includes the following core services:

- **IAM Service** – User authentication and role-based access control  
- **Buildings Service** – Management of buildings, floors, apartments, and residents  
- **Tickets System** – Residents can create and manage maintenance issues or requests  
- **Community Feed (per building)** – Internal posts and updates within each building (implemented as a separate microservice)
